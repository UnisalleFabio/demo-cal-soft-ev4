# Issues de ejemplo para GitHub

Estos archivos permiten simular en clase como se veria el seguimiento del modulo en GitHub Issues.

## Uso sugerido en clase

1. Mostrar primero [metricas-y-seguimiento.md](../metricas-y-seguimiento.md).
2. Abrir luego uno de los issues de ejemplo.
3. Relacionar el issue con un caso de prueba y con una posible correccion en un pull request.
4. Cerrar mostrando el workflow de GitHub Actions como evidencia del estado tecnico.

## Nota

Los archivos no crean issues reales por si solos. Quedan como material listo para copiar o cargar cuando el ejemplo se publique en GitHub.
