# Gestion formal de pruebas del ejemplo

Este directorio deja el ejemplo listo para mostrar una gestion mas formal de pruebas.

## Herramientas sugeridas

- `Kiwi TCMS`: opcion libre recomendada para una gestion formal de pruebas accesible a estudiantes.
- `TestLink`: opcion libre y clasica para planes, casos y ejecuciones.

## Material incluido

- `casos-prueba-formales.csv`: catalogo de casos con estructura de herramienta de testing.
- `ejecuciones-prueba.csv`: corrida de ejemplo con estado y observaciones.

## Alcance real de esta integracion

El workspace no tiene una instancia externa de TestLink o Kiwi TCMS corriendo, por lo que aqui no existe una sincronizacion viva. Lo que si queda listo es el insumo estructurado para:

1. mostrar en clase como se formaliza un plan de pruebas;
2. cargar manualmente o adaptar a una herramienta real;
3. relacionar casos, ejecuciones, hallazgos y evidencia.
